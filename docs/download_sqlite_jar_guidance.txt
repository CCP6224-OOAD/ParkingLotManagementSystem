IMPORTANT: SQLite JDBC Driver Required
=====================================

Please download the SQLite JDBC driver manually:

1. Visit: https://repo1.maven.org/maven2/org/xerial/sqlite-jdbc/3.45.0.0/
2. Download: sqlite-jdbc-3.45.0.0.jar
3. Place the JAR file in this folder (lib/)

Alternative download locations:
- https://github.com/xerial/sqlite-jdbc/releases
- Maven Central: https://search.maven.org/artifact/org.xerial/sqlite-jdbc

The file size should be approximately 7-8 MB.

After downloading, verify the file exists:
ls -lh lib/sqlite-jdbc-3.45.0.0.jar
